Progetto: Client HTTP

üìò Teoria

Client HTTP: tokio runtime, reqwest, async/await, JSON parsing con serde, ProgressBar.

üîß Procedura

  1. Dipendenze: tokio, reqwest
  2. Async fetch function
  3. UI con ProgressBar

üíª Codice

use tokio::runtime::Runtime;
use reqwest;

async fn fetch_url(url: &str) -> Result> {
    let response = reqwest::get(url).await?;
    Ok(response.text().await?)
}

// In GTK callback:
let runtime = Runtime::new().unwrap();
runtime.spawn(async {
    let body = fetch_url("https://api.example.com").await.unwrap();
    println!("{}", body);
});
