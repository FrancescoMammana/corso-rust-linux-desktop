Strutture di Controllo

üìò Introduzione Teorica

Rust offre strutture di controllo espressive:

  • if/else: Condizionali con valore di ritorno
  • loop: Ciclo infinito con break value
  • while: Ciclo condizionale
  • for: Iterazione su range e collezioni
  • match: Pattern matching potente ed esaustivo

üîß Procedura Passo-Passo

  1. if expression 
    let number = 6;
let result = if number % 2 == 0 { "pari" } else { "dispari" };
  2. loop con break value 
    let mut counter = 0;
let result = loop {
    counter += 1;
    if counter == 10 { break counter * 2; }
};
  3. for su range 
    for i in 1..=5 {
    println!("{}", i);
}
  4. match 
    match number {
    1 => println!("uno"),
    2..=5 => println!("da 2 a 5"),
    _ => println!("altro"),
}

üíª Codice Sorgente Commentato

fn main() {
    // IF EXPRESSION
    let x = 5;
    let y = if x > 0 { 10 } else { -10 };

    // LOOP
    let mut count = 0;
    let result = loop {
        count += 1;
        if count == 10 { break count; }
    };

    // WHILE
    while count > 0 {
        println!("{}", count);
        count -= 1;
    }

    // FOR
    for i in 1..=5 {
        println!("i = {}", i);
    }

    // MATCH
    let number = 7;
    match number {
        1 => println!("uno"),
        2 | 3 | 5 | 7 => println!("primo"),
        4 | 6 | 8 | 10 => println!("pari"),
        _ => println!("altro"),
    }
}

‚úèÔ∏è Esercizi Pratici

  1. FizzBuzz: Stampa numeri 1-100, "Fizz" se divisibile per 3, "Buzz" per 5, "FizzBuzz" per entrambi.
  2. Fibonacci: Genera primi 10 numeri Fibonacci con loop.
  3. Calcolatrice: Usa match per (+, -, *, /) su due numeri.
  4. Countdown: While loop che conta da 10 a 0.
