Strutture di Controllo

üìò Introduzione Teorica

Flusso controllo: if/else, loop, while, for, match.

  • if/else: Condizionali con valore
  • loop: Ciclo infinito con break value
  • while: Ciclo condizionale
  • for: Iterazione su range
  • match: Pattern matching esaustivo

üîß Procedura Passo-Passo

  1. if expression 
    let result = if x > 0 { 10 } else { -10 };
  2. loop con break value 
    let result = loop {
    counter += 1;
    if counter == 10 { break counter * 2; }
};
  3. for su range 
    for i in 1..=5 {
    println!("{}", i);
}

üíª Codice Sorgente Commentato

// IF EXPRESSION
fn main() {
    let x = 5;
    let y = if x > 0 { 10 } else { -10 };
    println!("{}", y);
}

// LOOP CON BREAK VALUE
fn main() {
    let mut count = 0;
    let result = loop {
        count += 1;
        if count == 10 { break count * 2; }
    };
    println!("Result: {}", result);
}

// FOR RANGE
fn main() {
    for i in 1..=5 {
        println!("i = {}", i);
    }
}

// MATCH
fn main() {
    let number = 3;
    match number {
        1 => println!("uno"),
        2..=5 => println!("da 2 a 5"),
        _ => println!("altro"),
    }
}

  1. FizzBuzz: Loop 1-100 con if/match.
  2. Fibonacci: While genera 10 numeri.
  3. Calcolatrice: Match per (+,-,*,/).
  4. Countdown: For loop decrescente.
