Sviluppo Manuale GUI

üìò Introduzione Teorica

Layout manuali con Box, Grid, Paned:

  • Box: Layout lineare H/V
  • Grid: Layout a griglia
  • Paned: Split ridimensionabile
  • Dynamic UI: Widget a runtime

üîß Procedura Passo-Passo

  1. Box Layout 
    let vbox = Box::new(Orientation::Vertical, 10);
vbox.pack_start(&widget, false, false, 0);
  2. Grid Layout 
    let grid = Grid::new();
grid.attach(&label, 0, 0, 1, 1);

üíª Codice Sorgente Commentato

use gtk::{Grid, Label, Entry, Button};

fn form_grid() {
    let grid = Grid::new();
    grid.set_row_spacing(10);
    grid.set_column_spacing(10);

    let nome_label = Label::new(Some("Nome:"));
    let nome_entry = Entry::new();
    grid.attach(&nome_label, 0, 0, 1, 1);
    grid.attach(&nome_entry, 1, 0, 1, 1);

    let email_label = Label::new(Some("Email:"));
    let email_entry = Entry::new();
    grid.attach(&email_label, 0, 1, 1, 1);
    grid.attach(&email_entry, 1, 1, 1, 1);

    let button = Button::with_label("Salva");
    grid.attach(&button, 0, 2, 2, 1);
}
