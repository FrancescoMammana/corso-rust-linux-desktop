Widget e Eventi

üìò Introduzione Teorica

Widget avanzati di GTK:

  • Entry: Input testuale
  • TextView: Editor multilinea
  • CheckButton: Checkbox
  • RadioButton: Opzioni multiple
  • Signals: Gestione eventi

üíª Codice Sorgente Commentato

use gtk::prelude::*;
use gtk::{Entry, TextView, CheckButton};

fn widgets_demo() {
    let entry = Entry::new();
    entry.connect_changed(|e| {
        println!("Text: {}", e.text());
    });

    let textview = TextView::new();
    let buffer = textview.buffer().unwrap();
    buffer.set_text("Contenuto");

    let check = CheckButton::with_label("Accetto");
    check.connect_toggled(|c| {
        println!("Toggled: {}", c.is_active());
    });
}
