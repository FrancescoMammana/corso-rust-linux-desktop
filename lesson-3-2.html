Sviluppo con Glade Designer

üìò Introduzione Teorica

Glade √® il designer visuale per GTK che genera file XML:

  • Drag & drop UI: Progetta interfacce visivamente
  • File .glade XML: Leggibile e versionabile
  • Builder pattern: Carica UI programmaticamente
  • Properties panel: Configura widget senza codice
  • Signal handlers: Collega eventi visualmente
  • Rapid prototyping: Crea UI in minuti

  1. Installa Glade 
    sudo apt install glade -y
  2. Crea interfaccia visuale

    Apri Glade ‚Üí New ‚Üí GTK+ 3.0 ‚Üí Aggiungi Window, Box, Button

  3. Salva file .glade 
    File ‚Üí Save As ‚Üí ui/main.glade

let window: Window = builder.object("main_window").expect("window");

üíª Codice Sorgente Commentato

use gtk::{Builder, Window, prelude::*};

fn main() {
    gtk::init().unwrap();

    // Carica UI da file Glade
    let builder = Builder::from_string(include_str!("../ui/main.glade"));

    let window: Window = builder
        .object("main_window")
        .expect("Could not find main_window");

    let button = builder.object::("submit_btn")
        .expect("Could not find submit_btn");

    button.connect_clicked(|_| {
        println!("Button clicked!");
    });

    window.show_all();
    gtk::main();
}
