Introduzione a GTK-rs

üìò Introduzione Teorica

GTK-rs √® il binding ufficiale Rust per GTK+:

  • Widget-based: Componenti UI riusabili
  • Signal system: Sistema eventi robusto
  • Container layout: Box, Grid, Paned
  • Native look: Si integra con GNOME
  • Thread-safe: Gestione sicura concorrenza

üîß Procedura Passo-Passo

  1. Setup GTK 
    cargo new hello_gtk
# Aggiungi Cargo.toml:
# [dependencies]
# gtk = { version = "0.18", features = ["v3_24"] }
  2. Application pattern 
    use gtk::prelude::*;
use gtk::Application;

fn main() {
    let app = Application::new();
    app.connect_activate(build_ui);
    app.run();
}

üíª Codice Sorgente Commentato

use gtk::prelude::*;
use gtk::{Application, ApplicationWindow, Button};

fn main() {
    let app = Application::builder()
        .application_id("com.example.HelloGTK")
        .build();

    app.connect_activate(build_ui);
    app.run();
}

fn build_ui(app: &Application) {
    let window = ApplicationWindow::builder()
        .application(app)
        .title("Hello GTK")
        .default_width(400)
        .default_height(300)
        .build();

    let button = Button::with_label("Cliccami!");
    button.connect_clicked(|_| {
        println!("‚úì Clicked!");
    });

    window.add(&button);
    window.show_all();
}

  1. Hello World: Finestra con bottone.

  3. Resize Window: Bottone ingrandisce finestra.
  4. Color Changer: 3 bottoni cambiano colore.
  5. Multi-Window: Apri finestre multiple.
  6. Challenge: Calcolatrice base (+,-,*,/).
