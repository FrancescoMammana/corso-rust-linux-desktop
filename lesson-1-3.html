Strumenti Aggiuntivi (GTK)

üìò Introduzione Teorica

Per sviluppare GUI desktop Linux con GTK:

  • GTK+ 3.0: Librerie grafiche
  • pkg-config: Tool per librerie
  • Glade: Designer visuale

üîß Procedura Passo-Passo

  1. Installa GTK 
    sudo apt install libgtk-3-dev libglib2.0-dev -y
  2. Installa pkg-config 
    sudo apt install pkg-config -y
  3. Verifica GTK 
    pkg-config --modversion gtk+-3.0
  4. Installa Glade 
    sudo apt install glade -y
  5. Test progetto GTK 
    cargo new test_gtk
cd test_gtk

    Aggiungi al Cargo.toml:

    [dependencies]
gtk = { version = "0.18", features = ["v3_24"] }
    cargo build

üíª Codice Sorgente Commentato

Cargo.toml:

[package]
name = "test_gtk"
version = "0.1.0"
edition = "2021"

[dependencies]
gtk = { version = "0.18", features = ["v3_24"] }
glib = "0.18"

src/main.rs:

use gtk::prelude::*;
use gtk::{Application, ApplicationWindow, Button};

fn main() {
    let app = Application::builder()
        .application_id("com.example.TestGTK")
        .build();

    app.connect_activate(|app| {
        let window = ApplicationWindow::new(app);
        let button = Button::with_label("Cliccami!");
        button.connect_clicked(|_| {
            println!("Clicked!");
        });
        window.add(&button);
        window.show_all();
    });

    app.run();
}

Esegui:

cargo run
