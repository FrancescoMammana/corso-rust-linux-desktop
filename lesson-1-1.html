Installazione di Rust su Debian

üìò Introduzione Teorica

Rust √® un linguaggio di programmazione moderno che offre prestazioni elevate, sicurezza della memoria senza garbage collector e un eccellente supporto per la concorrenza. √à particolarmente adatto per lo sviluppo di applicazioni desktop su Linux grazie a:

  • Performance native: Rust compila direttamente in codice macchina, offrendo prestazioni comparabili a C/C++
  • Sicurezza della memoria: Il sistema di ownership previene bug comuni come null pointer dereference e data races
  • Ecosistema ricco: Cargo e crates.io forniscono accesso a migliaia di librerie
  • Interoperabilit√†: Rust si integra perfettamente con librerie C come GTK

Per sviluppare applicazioni desktop su Debian, inizieremo installando Rust tramite rustup, il gestore ufficiale delle toolchain Rust.

üîß Procedura Passo-Passo

  1. Aggiorna il sistema 
    sudo apt update && sudo apt upgrade -y
    Questo assicura che tutti i pacchetti di sistema siano aggiornati.
  2. Installa curl (se non presente) 
    sudo apt install curl build-essential -y
    curl serve per scaricare rustup, mentre build-essential include gcc e altri strumenti necessari per compilare.
  3. Scarica e installa rustup 
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
    Durante l'installazione, seleziona l'opzione 1 (installazione predefinita).
  4. Configura il PATH 
    source $HOME/.cargo/env
    Questo comando carica le variabili d'ambiente. Per renderle permanenti, il comando viene aggiunto automaticamente a ~/.bashrc.
  5. Verifica l'installazione 
    rustc --version
cargo --version
rustup --version
    Dovresti vedere le versioni installate di ciascun tool.

‚úèÔ∏è Esercizi Pratici

  1. Crea il tuo primo progetto

    Usa cargo new mio_primo_progetto per creare un nuovo progetto. Naviga nella directory e esegui cargo run.

  2. Modifica il messaggio

    Apri src/main.rs e modifica il messaggio di benvenuto per includere il tuo nome. Esempio: "Benvenuto in Rust, [TuoNome]!"

  3. Aggiungi informazioni di sistema

    Usa std::env::consts::OS e std::env::consts::ARCH per stampare sistema operativo e architettura.

  4. Compila in release

    Esegui cargo build --release e nota la differenza di dimensione tra i binari in target/debug/ e target/release/.

  5. Verifica gli aggiornamenti

    Esegui rustup update per verificare se ci sono aggiornamenti disponibili per la toolchain Rust.

Problema: "rustc: command not found"

Soluzione: Il PATH non √® configurato correttamente. Esegui:

source $HOME/.cargo/env

Se il problema persiste, aggiungi manualmente questa riga a ~/.bashrc:

Poi ricarica la configurazione: source ~/.bashrc

Problema: Errori durante l'installazione di rustup

Soluzione: Assicurati di avere curl e build-essential installati:

sudo apt install curl build-essential -y

Problema: Permessi negati su ~/.cargo

Soluzione: Verifica i permessi della directory:

ls -la ~/.cargo
# Se necessario, correggi i permessi:
chmod -R u+rw ~/.cargo
