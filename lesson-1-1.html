Installazione Rust su Debian

üìò Introduzione Teorica

Rust √® un linguaggio moderno, sicuro e performante per applicazioni desktop Linux.

  • rustup: Gestore toolchain Rust
  • cargo: Build system e package manager
  • rustc: Compilatore Rust
  • Standard library: Libreria inclusa

üîß Procedura Passo-Passo

  1. Aggiorna sistema 
    sudo apt update
sudo apt upgrade -y
  2. Installa dipendenze 
    sudo apt install build-essential curl git -y
  3. Installa Rust con rustup 
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
  4. Configura PATH 
    source $HOME/.cargo/env
  5. Verifica installazione 
    rustc --version
cargo --version
  6. Primo progetto 
    cargo new hello_rust
cd hello_rust
cargo run

üíª Codice Sorgente Commentato

// File: src/main.rs
fn main() {
    println!("Hello, world!");
}

Compilazione:

cargo build
cargo run

Comandi cargo essenziali:

cargo new nome_progetto    # Nuovo progetto
cargo build               # Compila (debug)
cargo build --release     # Compila (release)
cargo run                 # Compila + esegue
cargo check               # Verifica veloce
cargo test                # Esegue test
cargo clean               # Rimuove build
