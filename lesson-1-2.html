Configurazione VS Code

üìò Introduzione Teorica

Visual Studio Code √® l'editor ideale per Rust:

  • rust-analyzer: Language server completo
  • CodeLLDB: Debugger integrato
  • Even Better TOML: Syntax per Cargo.toml

üîß Procedura Passo-Passo

  1. Installa VS Code 
    wget -qO- https://packages.microsoft.com/keys/microsoft.asc | gpg --dearmor > packages.microsoft.gpg
sudo install -o root -g root -m 644 packages.microsoft.gpg /etc/apt/trusted.gpg.d/
sudo sh -c 'echo "deb [arch=amd64] https://packages.microsoft.com/repos/code stable main" > /etc/apt/sources.list.d/vscode.list'
sudo apt update
sudo apt install code -y
  2. Installa rust-analyzer

    Extensions (Ctrl+Shift+X) ‚Üí Cerca "rust-analyzer" ‚Üí Install

  3. Configura settings.json 
    {
    "rust-analyzer.checkOnSave.command": "clippy",
    "rust-analyzer.cargo.allFeatures": true,
    "editor.formatOnSave": true
}

üíª Codice Sorgente Commentato

File: .vscode/settings.json

{
    "rust-analyzer.checkOnSave.command": "clippy",
    "rust-analyzer.cargo.allFeatures": true,
    "editor.formatOnSave": true,
    "[rust]": {
        "editor.defaultFormatter": "rust-lang.rust-analyzer"
    }
}

File: .vscode/launch.json

{
    "version": "0.2.0",
    "configurations": [
        {
            "type": "lldb",
            "request": "launch",
            "name": "Debug",
            "program": "${workspaceFolder}/target/debug/${workspaceFolderBasename}",
            "args": [],
            "cwd": "${workspaceFolder}"
        }
    ]
}
